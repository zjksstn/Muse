名称释义： 
项目名 Muse，deepseek译为灵感之源，这是一款本地mp3播放器。可用于mp3或mp3列表文件的播放与编辑。

mp3列表文件：程序默认支持的mp3列表文件后缀为.lst。可使用内置的编辑工具或任意一款文本编辑软件进行生成和编辑。
   
            列表文件的内容：按行输入mp3的绝对路径，采用utf-8编码保存为lst格式即可。

MP3工具：位于播放器封面图片右上的按钮，点击可打开内置的工具窗口。   
  
  本程序只支持mp3一种类型的播放，其余类型可通过内置的 mp3编辑工具转换成 mp3类型。
    
    格式转换需先安装ffmpeg，官网地址：https://ffmpeg.org/download.html
  
  程序内置 mp3编辑工具，可以对mp3文件的元数据进行编辑修改
	
  编辑工具可编辑的元数据包括封面、标题、专辑、演唱者四种，支持全曲库批量变更，并可序列化标题项目

  编辑工具可以通过Googls在线搜索、匹配 mp3文件的封面图片，但需事先自行解决科学上网问题。


曲库：本mp3工具所定义的mp3文件默认存放目录，可以通过变更曲库路径按钮，批量编辑不同目录下的mp3文件

可按用户选择的 .lst 文件增量变更或同步车载U盘曲目

	增量变更：根据lst文件名列表，检查、比较u盘中的曲目，如果一致则跳过，如果缺失或时间戳有更新则在u盘增加曲目

	同步：增量变更同时删除存在于u盘目录但不存在于lst列表内容的曲目
